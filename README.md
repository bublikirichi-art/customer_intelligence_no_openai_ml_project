# Customer Intelligence — No-OpenAI ML Ensemble

This project uses **no OpenAI API, no LLM, no external paid service**.

It uses all 400 labeled `train.csv` rows and builds a classical ML ensemble:

- character n-grams
- word n-grams
- title n-grams
- bilingual keyword/rule count features
- LinearSVC
- LogisticRegression
- 5-fold cross-validation
- separate classifiers for:
  - `recommended`
  - `reviewer_segment`

## GitHub run

Upload this project to the repository root so paths look like:

```text
data/train.csv
data/test.csv
src/build_no_openai_ensemble.py
.github/workflows/run_no_openai_ensemble.yml
requirements.txt
```

Then run:

```text
Actions → Run No-OpenAI ML Ensemble → Run workflow
```

No secrets are required.
